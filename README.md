# Vanie

Portfolio personnel et site web de Kirlinger Jeune — développeur web passionné par la culture haïtienne et les technologies modernes.

## 🌐 Voir le site

Le site est déployé automatiquement via GitHub Pages à chaque push sur `main`.

**URL** : [https://kirlinger.github.io/Vanie/](https://kirlinger.github.io/Vanie/)

## 🛠️ Technologies

- HTML5 sémantique
- CSS3 (responsive, animations, variables CSS)
- JavaScript vanilla
- GitHub Pages (déploiement automatique)

## 📁 Structure

```
Vanie/
├── index.html              # Page principale
├── css/
│   └── style.css           # Styles du site
├── js/
│   └── script.js           # Interactions (menu, scroll)
├── .github/
│   └── workflows/
│       └── deploy.yml      # Déploiement GitHub Pages
└── README.md
```
